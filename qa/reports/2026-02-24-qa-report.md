## Web-Only Issues Found

These are NOT bugs — they are expected web platform limitations:

1. **`Alert.alert()` not supported on web** — Affects: trip deletion (Test 17)
2. **`Alert.prompt()` not supported on web** — Affects: connect flow (Test 29), add to itinerary (Test 33)
3. **`ActionSheet` / overflow menus** — Affects: block (Test 30), report (Test 31)
4. **`DateTimePicker` not supported on web** — Onboarding can't be completed via UI (worked around via DB)

### Recommendation
Replace `Alert.alert()` and `Alert.prompt()` with custom modal components that work cross-platform. This would also improve the UX on iOS (native alerts look dated).
